# HotellBooking__Form
Luxury Hotel Booking form
Project Overview

The Luxury Hotel Booking form is a fully responsive, interactive web application for booking hotel rooms.
It features:

Animated background carousel of hotel images.

Welcome page with gradient title and enter button.

Booking page with room selection, form validation, and automatic amount calculation.

Success page showing booking details after submission.

Responsive design for desktop, tablet, and mobile devices.

Features

Animated Background – Continuous horizontal scroll of hotel images.

Welcome Page – Elegant cursive heading with gradient text.

Room Selection – Clickable room images; selected room highlights with border.

Booking Form – Collects name, email, phone, check-in/out dates; validates input.

Automatic Pricing – Displays room price dynamically when a room is selected.

Success Page – Shows the confirmed booking details after submission.

Responsive Design – Works on all screen sizes (mobile, tablet, desktop).

Technology Stack

HTML5 – Structure of the pages

CSS3 – Styling, animations, gradient effects, responsiveness

JavaScript (Vanilla JS) – Interactivity, form validation, dynamic updates

How to Use

Open index.html in any modern browser.

On the Welcome Page, click “Enter Booking” to go to the booking form.

Select a room type by clicking on a room image.

Fill out your Full Name, Email, Phone Number, Check-in, and Check-out dates.

The Amount field will update automatically based on selected room.

Click “Book Now” to submit the form.

You will see a Success Page showing all your booking details.

Form Validation

All fields required – user cannot submit empty fields.

Phone number – must be exactly 10 digits.

Check-out date – must be after check-in date.

Alerts are displayed for invalid or missing inputs.

How to Run Locally
